# Data Model — 修复 Alpine 中文渲染乱码

## 实体与字段

### 渲染请求
- `source`: Mermaid Gantt 文本（含中文）。
- `output`: 输出路径或 Writer。
- `theme`: 主题设置。
- `font_path`: 显式字体文件路径。
- `font_env`: 环境变量 `GGM_FONT_PATH`。
- `timezone/weekend/excludes/includes`: 时间与工作日配置（保持现有格式）。

### 字体来源
- `path`: 使用的字体文件路径或标识（显式/环境/内置）。
- `readable`: 可读性检查结果（bool）。
- `glyph_coverage`: 是否覆盖所需中文字符的检查结果（可通过加载成功与否近似）。
- `error`: 不可读或缺字时的错误信息。

### 渲染输出
- `bytes`: PNG 字节结果。
- `output_path`: 最终输出路径（若写入文件）。
- `diagnostics`: 与字体相关的提示/错误（用于返回或日志）。

## 关系
- 渲染请求 **选择** 字体来源（显式 > 环境 > 回退）。
- 渲染输出 **引用** 字体来源的诊断结果，用于报告或测试断言。

## 规则与验证
- 字体路径不存在/不可读 → 立即返回包含路径的错误。
- 字体加载失败（损坏/缺字）→ 返回错误而非生成乱码。
- 未提供任何可用字体时，不生成输出文件/字节，返回可诊断信息。
